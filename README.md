# テスト用 ローカル

### 改訂履歴
- 2023/08/15: 初版

### 概要
- テスト用の環境
- サンドボックスと名称についているが常時サンドボックスではない
  - サンドボックス環境でのテストとローカル環境でのテストを行うコンテナ

### ファイル命名規則
- 原則キャメルケース
  - Makefileやpackage-lock.jsonなど既に決まっているファイルは除外

### ファイル内命名規則
- 原則キャメルケース
  - クラス、enum、typeはパスカルケース

### コード規約
- 型定義
  - インターフェイスとtypeは用途が被る為、本リポジトリではtypeを使用
